# AI Executive Copilot Prompts

Collection of prompt systems designed to use AI as an operational copilot.

## Prompt 1: Executive Brief Generator
Prompt:
Summarize these meeting notes into:
- key decisions
- risks
- follow-up actions
- executive talking points

Use Case:
Accelerates meeting preparation.

---

## Prompt 2: Calendar Conflict Advisor
Prompt:
Review these competing meetings and recommend prioritization options with tradeoffs.

Use Case:
Supports schedule triage.

---

## Prompt 3: Stakeholder Drafting Assistant
Prompt:
Draft a concise stakeholder update covering progress, blockers, and next steps.

Use Case:
Improves executive communication speed.

---

## Prompt 4: Travel Risk Planner
Prompt:
Review this itinerary and identify potential risks, dependencies, and backup options.

Use Case:
Supports executive travel planning.
